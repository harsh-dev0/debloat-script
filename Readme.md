**Complete all-in-one PowerShell script**, bundling, so I can run it on any new Windows install in one shot.

This script performs:

* Disable OneDrive completely
* Remove OneDrive from File Explorer
* Disable XBox services and apps
* Disable Widgets
* Disable Microsoft Store
* Disable Telemetry
* Disable Edge background tasks
* Disable MongoDB auto start
* Disable Microsoft Office background service
* Disable bloat services
* Cleanup registry startup items
* Revert known folders to local
* System performance optimizations
